# Machine Learning Fundamentals Interview Questions

# Basic Level

### 1. What is Machine Learning?

Machine Learning is a subset of AI that enables systems to learn patterns from data and make predictions without explicit programming.

---

### 2. Difference between AI, ML and DL?

AI:
Making machines intelligent.

ML:
Learning patterns from data.

DL:
Subset of ML using deep neural networks.

---

### 3. What are the main types of Machine Learning?

- Supervised Learning
- Unsupervised Learning
- Reinforcement Learning

Additional:

- Semi-Supervised Learning
- Self-Supervised Learning

---

### 4. What is a feature?

An input variable used for prediction.

Example:

Age, Salary

---

### 5. What is a target variable?

The output variable that the model predicts.

---

### 6. What is training data?

Data used to train the model.

---

### 7. What is testing data?

Data used to evaluate the model.

---

### 8. What is a model?

A mathematical representation learned from data.

---

### 9. What is inference?

Using a trained model to make predictions.

---

### 10. What is supervised learning?

Learning from labeled data.

---

# Intermediate Level

### 11. Difference between Classification and Regression?

Classification:

Predicts categories.

Regression:

Predicts continuous values.

---

### 12. What is overfitting?

Model memorizes training data and performs poorly on unseen data.

---

### 13. What is underfitting?

Model fails to learn patterns from data.

---

### 14. What is bias?

Error due to overly simplistic assumptions.

---

### 15. What is variance?

Sensitivity of model to training data changes.

---

### 16. Explain Bias-Variance Tradeoff.

Balancing underfitting and overfitting.

---

### 17. Why is feature scaling important?

Prevents large-valued features from dominating smaller-valued features.

Important for:

- KNN
- SVM
- Gradient Descent

---

### 18. What is cross-validation?

Technique used to evaluate model stability by training on multiple subsets.

---

### 19. What is feature engineering?

Creating useful features from raw data.

---

### 20. What is data leakage?

Using future information during training.

Causes unrealistic performance.

---

# Advanced Level

### 21. Why does ML require large datasets?

More data helps models generalize better.

---

### 22. What is the curse of dimensionality?

Performance degradation when feature count becomes very high.

---

### 23. What is regularization?

Technique to reduce overfitting.

Examples:

- L1
- L2

---

### 24. What is concept drift?

Data distribution changes over time.

---

### 25. What is class imbalance?

One class dominates another.

Example:

99% Non-Fraud
1% Fraud

---

### 26. Difference between Parametric and Non-Parametric Models?

Parametric:

Fixed parameters.

Example:
Linear Regression

Non-Parametric:

Flexible parameters.

Example:
KNN

---

### 27. What is Explainable AI?

Methods that help understand model decisions.

Examples:

SHAP
LIME

---

### 28. Why can high accuracy be misleading?

Imbalanced datasets.

Example:

99% Non-Fraud
1% Fraud

Predicting all non-fraud gives 99% accuracy.

---

### 29. What is Generalization?

Ability to perform well on unseen data.

---

### 30. What is the ultimate goal of Machine Learning?

Learn patterns that generalize to unseen data and make reliable predictions.
